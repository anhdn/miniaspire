
## Getting started
```
cd my-project
yarn install
yarn start
```

Then open `http://localhost:8080/` to see your app.

### Deployment
After `npm run build` finished, run

```
yarn serve
```

### Technology stack:
- React v16.8
- Next.js v9
- Immutable.js
- Styled-Components
- Bootstrap v4
- Sass
- ESLint
- Prettier
- Webpack
